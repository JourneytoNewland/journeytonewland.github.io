随手记 - 一些信息源

2019-04-29

我们在追逐“成就”的过程中，最稀缺的资源就是时间，任何人想要有杰出的表现，一定要“珍惜时间”。

学习意愿 * 学习能力 * 学习方法 = 成长速度

贵精不贵多。请保留你自己最后的裁决。

  • 算法实现 - Python：我估计我是不会再跑一跑了
  • GitHub 的边框树形插件：一个美好的插件
  • 当···时发生了什么？：知其然，知其活动轨迹
  • 开源大数据平台Greenplum中文社区：用过。
  • 十年的标签库建设经历，我得到了什么启示？ ：可以对标客户标签库的定位，整体与个人理解基本一致，可以多看两眼
  • 交互式图解人工智能-OKAI：科普，以后写材料截图，截表述用，还是可以的：）
  • Plotly Express ：作为一个新的 Python 可视化库，可以玩玩
  • Requests-HTML：HTML Parsing for Humans
  • 全栈增长工程师指南
  • 中国大学MOOC、学堂在线、网易云课堂、好大学在线、爱课程 MOOC 课程下载：就是（差）需要分点时间认真看看，练练
  • Learn python3 in one picture：主要是现在好学的原因，实例都在py3-in-one-pic.ipynb文件里。抄都不用抄。
  • 宋词智能分词数据应用：漂亮
  • Vue element admin： 是一个后台前端解决方案，它基于 Vue 和 Element-ui实现。对 Vue 感兴趣的，自己用来做做 Demo 用也可以
  • AI 算法工程师手册
  • 一个个人博客的关于AI的资料下载：人家的，一个人像个队伍：）
  • Jeecg-Boot 快速开发平台(前后端分离版本)：某天深夜调了我半宿（我的电脑真的要太响了），没有全部调通。不过，开源，目前还有更新，理念不错，有空再调一下下。
  • 腾讯云的在线学习中心

我最近老感觉，我是不是应该搞个纸质笔记本来记笔记，遥想当年一本纸质笔记本里记得东东，让自己好几年都感觉如鱼得水。要不然，复制粘贴太快了，快得来不及完全进入脑子。要深刻反省！
